Product Changelog:

  • SkewT Overhaul, Hodographs Only - Only available after the 2025/04/19 06z run.
  • 1-Hour Dewpoint Change, new locations for SkewTs & Point Data - Only available after the 2025/04/16 00z run.
  • Max CAPE, CIN, and streamlines on windmaps - Only available after the 2025/04/12 00z run.
  • 1-Hour Temp Change (all levels) - Only available after the 2025/04/11 03z run.
  • 700mb/500mb Heights, RH (all levels) - Only available after the 2025/04/04 00z run. Additionally, the cloud cover maps were modified and had the ends of their colormaps swapped.
  • 4-Panel Cloud Cover, improved Cloud Cover - Only available after the 2025/04/03 03z run.
  • Minor bug fix - Before the 2025/03/31 03z run, the upper air plots displayed EST time was actually in EDT. This is truly fixed now, promise!
  • Aloft temp colormap adjustments - Before the 2025/03/26 06z run, temperatures aloft were plotted on a poor colormap. This was adjusted in this run.
  • Upper air maps, improved MSLP contours, reflectivity colormap improvements - Only available after the 2025/03/25 03z run. Additionally, text displaying EST on weathermaps before this run is actually in EDT. However, upper air plots still display EDT until the 03/31 03z run.
  • Wind, wind gusts - Before the 2025/03/23 21z run, the "wind gusts" product was not available and "winds" actually plotted wind gusts.
  • Helicity Tracks, 1-Hour Snowfall, Cloud Cover, values plotted at airports - Only available after the 2025/03/23 06z run
  • 1-Hour Precip - Only available after the 2025/03/23 03z run
